📚 Quick Integration Guide

Use these keys in your application:

// Initialize PayMe Gateway
const payme = new PayMe({
  publishableKey: 'pk_test_YOUR_STRIPE_PUBLIC_KEY_HERE...',
  secretKey: 'sk_test_YOUR_STRIPE_SECRET_KEY_HERE...'
});

// Create a payment
const payment = await payme.createPayment({
  amount: 1000,
  currency: 'INR',
  description: 'Order #123'
});
