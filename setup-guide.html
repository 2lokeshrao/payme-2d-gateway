Email Configuration

Configure email settings to send activation codes to customers:

Gmail Configuration (Recommended)

Setting Value
SMTP Host smtp.gmail.com
SMTP Port 587
SMTP Username your-email@gmail.com
SMTP Password Gmail App Password (16 characters)

How to Get Gmail App Password

  1. Go to Google Account Settings
  2. Click "Security"
  3. Enable "2-Step Verification"
  4. Click "App Passwords"
  5. Select "Mail" and "Other (Custom name)"
  6. Enter "PayMe Gateway"
  7. Click "Generate"
  8. Copy the 16-character password
  9. Use this password in SMTP settings
‚ö†Ô∏è Important:

Use Gmail App Password, NOT your regular Gmail password. Regular passwords won't work for SMTP.
