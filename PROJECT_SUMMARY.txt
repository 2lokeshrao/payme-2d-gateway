================================================================================
                          BUSINESS MODEL
================================================================================

REVENUE STREAMS:
1. Direct Sales: Admin sells codes directly to users
2. Reseller Sales: Resellers sell codes with commission
3. Subscriptions: Recurring revenue from active plans

COMMISSION STRUCTURE:
• Admin sets commission rate per reseller (e.g., 20%)
• When code is activated, commission is calculated automatically
• Example: $100 plan × 20% = $20 to reseller, $80 to admin
• Commission tracked as "pending" until payout processed

CODE-BASED ACTIVATION:
• Users purchase activation codes from admin or resellers
• Each code activates a specific subscription plan
• Subscription duration starts immediately upon activation
• Service access expires when subscription period ends
• Renewal requires new activation code purchase

================================================================================
                          USAGE WORKFLOW
================================================================================

STEP 1: ADMIN CREATES PLAN
Admin → Plan Management → Create Plan
• Name: "Monthly Pro"
• Price: $60
• Duration: 30 days
• Features: List of features

STEP 2: ADMIN CREATES RESELLER
Admin → Reseller Management → Add Reseller
• Name, Email, Phone
• Commission Rate: 20%
• Assign Plans: Select which plans reseller can sell
• Generate login credentials

STEP 3: RESELLER GENERATES CODE
Reseller → Login → Dashboard → Generate Code
• Select plan (from allowed plans)
• Enter customer email (optional)
• Generate code: PM2D-XXXX-XXXX-XXXX
• Send code to customer

STEP 4: CUSTOMER ACTIVATES
Customer → Activation Page → Enter Code
• Enter 16-character code
• Click "Activate Subscription"
• Subscription starts immediately
• Commission calculated automatically

STEP 5: COMMISSION TRACKING
System automatically:
• Calculates commission (Plan Price × Rate)
• Creates earning record (Status: Pending)
• Updates reseller statistics
• Admin processes payout later
