Two Business Models

Model 2: Central Gateway with Resellers

Perfect for building a payment gateway business with resellers.

  • Create reseller accounts with commission rates
  • Resellers generate and sell activation codes
  • Revenue sharing between admin and resellers

üîå API Documentation

Authentication

All API requests require authentication using API key.

Get API Key:

  1. Login to User Panel
  2. Go to "API Settings"
  3. Click "Generate API Key"
  4. Copy API key

Authentication Header:

Authorization: Bearer YOUR_API_KEY

Create Payment Link

Endpoint: POST /api/create-payment-link.php

Request Body:

{ "amount": 1000, "description": "Product Purchase", "customer_email": "customer@example.com", "customer_name": "John Doe", "payment_methods": ["upi", "bank", "crypto"] }

Response:

{ "success": true, "payment_link": "https://your-domain.com/pay/abc123", "payment_id": "PAY123456", "expires_at": "2025-10-05 10:00:00" }

Verify Payment

Endpoint: POST /api/verify-payment.php

Request Body:

{ "payment_id": "PAY123456", "transaction_id": "UPI123456789" }

Response:

{ "success": true, "status": "completed", "amount": 1000, "transaction_id": "UPI123456789", "verified_at": "2025-10-04 10:30:00" }

Get Transaction Details

Endpoint: GET /api/get-transaction.php?id=TXN123456

Response:

{ "success": true, "transaction": { "id": "TXN123456", "amount": 1000, "status": "completed", "payment_method": "upi", "customer_email": "customer@example.com", "created_at": "2025-10-04 10:00:00", "completed_at": "2025-10-04 10:30:00" } }

üîß Troubleshooting

Common Issues and Solutions

Issue: Cannot Login to Admin Panel

Possible Causes:

  • Wrong username or password
  • Account locked due to failed attempts
  • Database connection error

Solutions:

  1. Verify credentials (default: admin/admin123)
  2. Reset password via database
  3. Check database connection in config file
  4. Clear browser cache and cookies

Issue: Emails Not Sending

Possible Causes:

  • Wrong SMTP credentials
  • Gmail blocking less secure apps
  • Firewall blocking SMTP port
  • Email in spam folder

Solutions:

  1. Verify SMTP settings
  2. Use Gmail App Password (not regular password)
  3. Check firewall settings
  4. Check spam folder
  5. Try different SMTP port (465 instead of 587)

Issue: Payment Details Not Showing

Possible Causes:

  • Payment configuration not saved
  • Database connection error
  • Browser cache issue

Solutions:

  1. Re-save payment configuration
  2. Check database connection
  3. Clear browser cache
  4. Try different browser
