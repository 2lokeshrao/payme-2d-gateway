üöÄ Quick Start

Installation

Step 1: Clone the Repository

git clone https://github.com/2lokeshrao/payme-2d-gateway.git cd payme-2d-gateway

Step 2: Setup Database

mysql -u root -p -e "CREATE DATABASE payme_gateway;" mysql -u root -p payme_gateway < database.sql

Step 3: Configure Database Connection

Edit config/database.php:

define('DB_HOST', 'localhost'); define('DB_NAME', 'payme_gateway'); define('DB_USER', 'your_username'); define('DB_PASS', 'your_password');

Step 4: Access the System

  • Homepage: http://your-domain.com
  • Admin Panel: http://your-domain.com/admin/login.html
  • User Panel: http://your-domain.com/user/login.html
