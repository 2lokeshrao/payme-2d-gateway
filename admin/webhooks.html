Webhook Documentation

ℹ️ How Webhooks Work:

Webhooks allow your application to receive real-time notifications when events occur in the payment gateway. When an event happens, we'll send a POST request to your webhook URL with event data.

Webhook Payload Example:

{ "event": "payment.success", "timestamp": "2025-10-04T10:35:42Z", "data": { "transaction_id": "TXN1234567890", "amount": 500, "currency": "INR", "payment_method": "UPI", "customer_email": "customer@example.com", "status": "success" }, "signature": "sha256_hash_of_payload" }

Verifying Webhook Signature:

// PHP Example $payload = file_get_contents('php://input'); $signature = $_SERVER['HTTP_X_WEBHOOK_SIGNATURE']; $secret = 'your_secret_key'; $expected_signature = hash_hmac('sha256', $payload, $secret); if (hash_equals($expected_signature, $signature)) { // Signature is valid, process the webhook $data = json_decode($payload, true); // Handle the event } else { // Invalid signature http_response_code(401); }
