1Ô∏è‚É£ JavaScript Integration (Easiest)

Step 1: Include the SDK

<script src="https://checkout.payme2d.com/v1/payme2d-sdk.js"></script>

Step 2: Initialize and Create Payment

<script>
const payme = new PayMe2D('pk_test_your_api_key');

// Create payment button
document.getElementById('payButton').addEventListener('click', () => {
    payme.checkout({
        amount: 1000,
        currency: 'INR',
        customer_email: 'customer@example.com',
        customer_phone: '+919876543210',
        description: 'Order #12345',
        onSuccess: function(payment) {
            console.log('Payment successful!', payment);
            alert('Payment completed! Transaction ID: ' + payment.transaction_id);
        },
        onError: function(error) {
            console.error('Payment failed:', error);
            alert('Payment failed: ' + error.message);
        },
        onClose: function() {
            console.log('Checkout closed');
        }
    });
});
</script>

3Ô∏è‚É£ REST API Integration (Backend)

Create Payment

POST https://api.payme2d.com/v1/payments
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "amount": 1000,
  "currency": "INR",
  "customer_email": "customer@example.com",
  "customer_phone": "+919876543210",
  "description": "Order #12345",
  "redirect_url": "https://yoursite.com/success",
  "webhook_url": "https://yoursite.com/webhook"
}

Response:
{
  "success": true,
  "transaction_id": "TXN123456789",
  "checkout_url": "https://checkout.payme2d.com/pay/TXN123456789",
  "status": "pending"
}

Get Payment Status

GET https://api.payme2d.com/v1/payments/TXN123456789
Authorization: Bearer YOUR_API_KEY

Refund Payment

POST https://api.payme2d.com/v1/refunds
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "transaction_id": "TXN123456789",
  "amount": 1000,
  "reason": "Customer requested refund"
}

4Ô∏è‚É£ WordPress Integration

Using Shortcode

[payme2d amount="1000" currency="INR" description="Product Purchase"]

Using PHP

<?php
// Add to functions.php
function payme2d_payment_button() {
    return '<button onclick="payWithPayMe2D()">Pay Now</button>
    <script>
    function payWithPayMe2D() {
        const payme = new PayMe2D("' . get_option('payme2d_api_key') . '");
        payme.checkout({
            amount: 1000,
            currency: "INR",
            customer_email: "' . wp_get_current_user()->user_email . '",
            onSuccess: function(payment) {
                window.location.href = "/thank-you";
            }
        });
    }
    </script>';
}
add_shortcode('payme2d_button', 'payme2d_payment_button');
?>
